# Pharmacy App

![Pharmacy App](https://github.com/franciscloudss/pharmacy-app/blob/master/src/assets/media/pharmacy-app.png?raw=true)

## 🤔 What is it?

App to enhance workflow between pharmacists from Farmacéuticos Asociados S.R.L. and doctors from Argentina.

## 💻 Can I see it live?

Of course! Visit the live version by clicking the following link: \
[farmaceuticos-asociados.web.app](https://farmaceuticos-asociados.web.app/)

## 🧱 What was your stack of technologies?

These are the technologies I used to create the project: \
✔ HTML \
✔ Styled Components \
✔ TypeScript \
✔ ReactJS \
✔ Redux \
✔ Google Sheets API
