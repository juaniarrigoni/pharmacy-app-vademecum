# Pharmacy App

![Pharmacy App](https://github.com/franarrigoni/pharmacy-app/blob/master/src/assets/pharmacy-app.png?raw=true)

## 🤔 What is it?

App to enhance workflow between pharmacists from Farmacéuticos Asociados S.R.L. and doctors from Argentina.

## 💻 Can I see it live?

Of course! Visit the live version by clicking the following link: \
[pharmacy-app.vercel.app](https://pharmacy-app.vercel.app/)

## 🧱 What was your stack of technologies?

These are the technologies I used to create the project: \
✔ HTML - CSS - Sass \
✔ TypeScript \
✔ ReactJS
